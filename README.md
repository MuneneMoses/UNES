# UNES Registry (Master List)





## Installation
1. Clone this repository.
```bash
git clone https://github.com/MuneneMoses/UNES
```
2. Open ```entity/index.html``` in browser.

## Demo
You can open the demo <a href="https://github.com/MuneneMoses/UNES/entity/presentation1.mp4">here</a>

## Contributing

- Fork it ( https://github.com/MuneneMoses/UNES.git )
- Create your feature branch (`git checkout -b my-new-feature`)
- Commit your changes (`git commit -am 'Add some feature'`)
- Push to the branch (`git push origin my-new-feature`)
- Create a new Pull Request

### Note
- If you want to make some changes in the CSS style. Don't edit the `.css` file, use the SCSS file instead.
- Since we use Nunjucks, if you want to make changes to the HTML structure, make sure you change the source file (Nunjucks) located in `src`.
